# Primordia

Simulation 2D locale de micro et macro-écosystèmes. Bactéries, virus, contamination, chaînes alimentaires, IA collective et individuelle, stats détaillées par spécimen, filtres visuels (sang, muscles, overlay bactérien), rendu cartoon procédural, contrôles temps-réel (pause / accélération / ralenti), zoom multi-échelle.

Zéro API externe. Tourne entièrement en local.

## Stack

- **Engine** : Godot 4.x
- **Langage** : GDScript
- **Rendu** : 2D + shaders custom (CanvasItem shaders)
- **Données** : tout en mémoire — pas de base de données

## Lancement

```bash
godot .
```

Ou ouvrir `project.godot` dans l'éditeur Godot 4.

## Ports

Aucun — application desktop standalone.
